# MyNAS

这个仓库主要用来记录近期在玩斐讯N1电视盒子的过程，同时也想通过这个方式提升一下自己的文案能力。可以说这个盒子实在是太好玩了，价格便宜，才80-110的价格，都可以在拼多多或者其他二手平台寻得。性能十分优秀，据说可以跟树莓派3B做对比。机身配置是千兆网口 + wifi + 蓝牙 + 两个USB2.0。可玩性非常高，目前主流的玩法有用来刷官改的电视盒子固件、小钢炮（NAS + 下载机）、SX05RE（游戏机）、各版本Linux。目前本人主要从事开发工作，主要玩的方向是私人NAS + web服务器。
